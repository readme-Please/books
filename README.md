## Estrutura

Todos os livros adicionados devem estar dentro de suas respectivas caterogrias, e contendo um arquivo .json seguindo o layout especificado a baixo.

### Estrutura arquivo Json;

```json
{
    "category": "",
    "language": "",
    "title": "",
    "author": "",
    "description": ""
}
```

### Estrutura de pastas
     ./books
        └ {nome Categoria}
             └{nome Categoria}
                ├{nome Livro}.pdf
                └{nome Livro}.json
